## SEO
Medium impact:

No canonical tag — <link rel="canonical"> is missing, which can cause duplicate-content issues if the site is reachable via both www and non-www.

No structured data (JSON-LD) — no Organization or LocalBusiness schema. This is a missed opportunity for rich results given this is a business site.

Low impact:

Generic alt text — "product image", "cyslogo", "partners" are vague. Descriptive alt text helps both accessibility and image search.
